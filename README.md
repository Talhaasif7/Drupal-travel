# Drupal Travel Guide Project 

Built to demonstrate Drupal 11 expertise and professional development workflows. 

### Key Technical Features: 
- **Entity Relationships**: Linked Activities to Destinations using Entity References. 
- **Advanced Views**: Created dynamic blocks with Contextual Filters for relational data display. 
- **Layout Builder**: Implemented component-based drag-and-drop page architecture. 
- **AI-Powered Content**: Integrated the **Drupal AI** suite to automate destination summaries.
    - **Secure Credentialing**: Used the **Key** module to manage OpenAI API secrets securely.
    - **Automated Pipelines**: Configured **AI Automators** to trigger GPT-4o-mini generations on entity save.
    - **Token Integration**: Utilized the **Token** module to pass dynamic node context into LLM prompts.
- **Configuration Management**: Fully exported all site architecture and AI settings to YAML for version-controlled deployments.
